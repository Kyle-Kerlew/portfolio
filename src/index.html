Twitter Bot

A Twitter bot I volunteered to create for a community on Twitter called #FitDevs. The bot is responsible for retweeting tweets containing the FitDevs hashtag. The bot also replies to tweets that engage with the community's #OneMillionMinutes hashtag, which aims for a million minutes of logged fitness activity. The tweeted replies contain the user's total fitness minutes, and the running total of all the minutes logged by the community. Note: The bot is no longer running due to changes in Twitter's API.

Hydroponic Data Dashboard

This is a cyberpunk themed dashboard for monitoring my personal hydroponic system at my home. The data shown is fetched from an ESP8266 microcontroller that is connected to a temperature sensor, a PH sensor, and a TDS sensor. The frontend polls the ESP8266 every 10 seconds for new data and updates the dashboard accordingly.
